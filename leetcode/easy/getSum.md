## 问题分析： 
不使用运算符 + 和-，计算两整数a 、b之和。

## 代码实现
```c
class Solution {
public:
    int getSum(int a, int b) {
    while(b){
            int c = a & b;  
            a = a ^ b;  
            b = c << 1;  
        }  
        return a;
    }
};
```
## 总结：
首先，判断b是否为0，如果是0，直接返回a值。如果不是0，开始内层循环，第一步做两个数的与操作，来实现标识应当进位的位置。然后，做一步异或操作将不考虑进位的加法结果求出来。最后一步，将c向左移一位，也就是向前进一位再把值传给b，这样，b的值就是要与a值进行相加的加数，以此类推，直到b的值为0，跳出循环，得到加法值。
